# destin
Destin est une sorte de mini moteur de jeu pour créer des livres dont on est le héros (ou potentiellement des romans visuels) et tout ce qui s'en rapproche.

Le but est surtout de s'entrainer à Qt mais ça commence à devenir fonctionnel.

Le but final est d'en faire une librairie utilisable pour des projets via Qt Creator mais comme je la change encore très souvent je préfère l'utiliser directement dans des projets associés destin (cf dans les autres repositories). C'est aussi pour ça qu'il y a dans le code les restes d'une application à base de tyranides qui n'est surement plus fonctionnelle et sera bientôt supprimée.
